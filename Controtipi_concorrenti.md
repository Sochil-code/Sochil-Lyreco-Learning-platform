# Mappatura controtipi concorrenti → catalogo Sochil/Lyreco

**Scopo:** per ogni prodotto concorrente, il controtipo Sochil (tra i 34 a catalogo Lyreco) più affine per funzione e settore. Dove Sochil non ha un prodotto uguale o simile → **NESSUN CONTROTIPO** (referenza fuori gamma).

**Metodo:** cross-check contro i formulati Sochil (funzione, pH, settore, formato). Profilo del concorrente indicativo: verifica sempre etichetta e SDS.

**Stato:** programma a lotti (22 aziende). Questo file cresce marca per marca; alimenta il database dell'app.

Legenda controtipi Sochil: Pavilux 2000, Ekosan, Sanson (pavimenti neutri) · Soquat, Igiesan RTU, Alcovir (disinfezione) · Grill, Fatpul, Ultranet, Net+, Green/Red Energy, Ergonet, Viaol, DeInk (sgrassanti) · Decrost, Limelock, Batsan, Boxi, Pavixan (anticalcare/disincrostanti) · Detersol VS, Du-Flash (vetri) · Ergonet, Viaol (deceranti) · Splende (lavaincera) · Presto, Desan, Super Action (bucato/tessuti) · Master Matic, Brilux (lavastoviglie) · Aersan (deodorante) · Xpluri, Plurel, Tek Ok (multi/food).
